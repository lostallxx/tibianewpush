# Product Notes

## O que este projeto e
Um treinador de push para Tibia, focado em repeticao, timing e leitura de SQM.

## O que este projeto nao precisa ser agora
- Um clone completo do Tibia.
- Um MMO.
- Uma engine nova antes da hora.

## Principios
- A mecanica de push atual e a referencia.
- Melhorar fidelidade sem destruir a simplicidade de iteracao.
- Toda feature nova deve responder: isso melhora treino ou prepara arquitetura?

## Proximos alvos tecnicos
- Tirar mais regras do componente principal.
- Criar estado de treino persistente.
- Estruturar cenarios em dados e nao em JSX.
- Introduzir testes para utilitarios de movimento e bloqueio.
